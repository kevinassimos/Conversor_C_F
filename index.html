Conversor de Temperatura¶

Introdução¶

Este projeto é um simples conversor de unidades de temperatura entre Celsius e Fahrenheit.

Funções: - celsius_para_fahrenheit: Converte de Celsius para Fahrenheit. - fahrenheit_para_celsius: Converte de Fahrenheit para Celsius.
